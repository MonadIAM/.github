## LIAM

Lambdius Identity & Access Management — an IAM platform designed as a reusable foundation for future applications.

> Status: Work in progress / Solo development / Dev mode only

### Motivation

The project is developed solo and only in spare time. There are no commercial goals at this point.

***What I aim to achieve:***
- A production-grade IAM core built from scratch — as systems design practice.
- A reusable foundation for future and personal applications.
- Hands-on experimentation with the chosen tech stack and architecture.
- Optional evolution into a SaaS product down the line — with no commitment on timeline or form.

### Structure

LIAM is organized as a set of microservices and micro-frontend applications — one per bounded context. Each repository contains its own documentation with setup, environment, and run instructions.

### Contributing

A personal pet project. External contributions are not expected at this stage, but each repository includes documentation for contributors.
